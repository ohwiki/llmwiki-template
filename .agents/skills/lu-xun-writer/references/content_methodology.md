# 鲁迅风格内容方法论

## 1. 选题方法论

### 选题原则

一条标准：**这件事里有没有「瞒和骗」？**

鲁迅一生最恨瞒和骗。凡是有人在粉饰、在调和、在合理化不合理的事，就是好选题。

### 选题质检：刺、真、痛

- **刺（穿透力）** — 这个选题能不能刺穿一层伪装？
- **真（事实感）** — 有没有具体的事实支撑，还是只有情绪？
- **痛（切肤感）** — 读者读完会不会觉得被说中了？

好选题至少占两项。只有情绪没有事实，是骂街。只有事实没有穿透力，是新闻。

### 选题颗粒度

要小。一篇杂文只需要一把刀。

❌ 太大：「AI的伦理问题」「社会不公」
✅ 刚好：「为什么被裁的人还要感谢公司」「AI生成的正能量文章为什么让人恶心」

## 2. 素材处理

### 找要害

拿到素材后第一件事：找要害。

不是找全貌。是找那个「大家都看到了但没人说」的点。那个点就是你的刀尖。

### 找看客

第二件事：找看客。

这件事里谁在围观？谁在沉默？谁在合理化？看客往往比施暴者更值得写。

### 找类比

第三件事：找一个历史或文学的类比。

鲁迅的杂文几乎每篇都有类比——用古讽今，用外讽中，用小讽大。好的类比让论点从「你的观点」变成「历史的规律」。

## 3. 内容节奏

### 鲁迅式节奏

不是波浪式推进。是「铺垫——铺垫——铺垫——一刀」。

**铺** — 平静地叙述事实。看似客观。读者跟着你看。
**铺** — 继续叙述。加入一点细节。读者开始觉得哪里不对。
**铺** — 再加一层。读者已经隐约知道你要说什么了。
**刀** — 最后一句。短。狠。准。读者愣住。

### 节奏规则

1. **铺垫不是废话。** 每一句铺垫都在蓄力。删掉任何一句，最后那一刀就没那么疼
2. **「然而」是转折之王。** 鲁迅最爱用的转折词。「然而」后面跟的永远是真相
3. **结尾比开头重。** 开头亮刀，结尾捅穿。读者记住的永远是最后一句

## 4. 类比工作法

### 类比原则

- 类比要有距离感。越远的类比越有力（用古代的事讽今天的事）
- 类比要精准。不是「像什么」，是「就是什么」
- 一个类比用到底。不要中途换

### 类比示例

| 要批判的现象 | 鲁迅式类比 |
|-------------|-----------|
| AI赋能 | 给镣铐换一个好听的名字 |
| 知识付费 | 卖假药的换了白大褂 |
| 正能量审查 | 从前叫「瞒和骗」，现在叫「正能量」 |
| 人人平等论 | 穷人和富人都有权睡在桥洞下面 |
| 裁员叫优化 | 杀人叫「送你上路」 |
