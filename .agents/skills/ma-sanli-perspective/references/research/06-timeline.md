## Agent 6: 人物时间线

[信息来源：Wikipedia + 公开传记资料]
[可信度：高——基本事实]

| 年份 | 事件 | 意义 |
|------|------|------|
| 1914 | 生于北京，回族，相声世家 | 父亲马德禄是「相声八德」之一 |
| 1929 | 15岁拜师周德山（艺名「周蛤蟆」） | 正式入行，属「寿」字辈 |
| 1930s | 在北京天桥、天津劝业场等地撂地演出 | 底层磨练期，直面最挑剔的观众 |
| 1940s | 逐渐成名，形成独特的「蔫哏」风格 | 风格定型期 |
| 1949 | 新中国成立，加入天津曲艺团 | 从民间艺人变为体制内演员 |
| 1950s | 创作《买猴》等讽刺作品 | 艺术巅峰期之一 |
| 1957 | 被打成右派 | 人生转折点——因为说了讽刺官僚的相声 |
| 1957-1979 | 下放劳动，被剥夺表演权利长达22年 | 人生最黑暗的时期，但从未放弃相声 |
| 1979 | 平反复出 | 65岁重返舞台 |
| 1980s-90s | 创作《逗你玩》《练气功》等经典 | 第二个艺术巅峰期，风格更加炉火纯青 |
| 1993 | 妻子去世（实际1984年成为鳏夫） | 晚年独居 |
| 2001 | 87岁正式退休，告别舞台 | 最后一场演出 |
| 2003.2.11 | 因癌症在天津去世，享年88岁 | 天津万人送别 |

### 思想转折点

1. **1957年被打成右派** — 从此明白：讽刺是有代价的。但他没有因此放弃讽刺，只是变得更隐晦
2. **1979年复出** — 65岁重新开始。从此他的幽默里多了一层通透——经历过最坏的，反而更轻松了
3. **晚年** — 从「说相声的人」变成「相声本身」。观众来看他，不是来听段子，是来看这个人
